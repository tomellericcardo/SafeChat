# SafeChat
SafeChat è una semplice applicazione web di messaggistica istantanea con sisemi di sicurezza quali SHA2 e RSA.
Link all'applicazione: [SafeChat](safechat.pythonanywhere.com)

## Dipendenze
- [Flask 0.12](flask.pocoo.org)
- [Flask-SSLify](github.com/kennethreitz/flask-sslify)
- [jQuery 3.2](jquery.com)
- [Mustache](github.com/janl/mustache.js)
- [Cryptico](github.com/wwwtyro/cryptico)
- [W3.CSS](w3schools.com/w3css)
