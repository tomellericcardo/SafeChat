# SafeChat
SafeChat è una semplice applicazione web di messaggistica istantanea con sisemi di sicurezza quali SHA2 e RSA.
Link all'applicazione: [SafeChat](safechat.pythonanywhere.com)

## Dipendenze
- [Flask 0.12](http://flask.pocoo.org)
- [Flask-SSLify](http://github.com/kennethreitz/flask-sslify)
- [jQuery 3.2](http://jquery.com)
- [Mustache](http://github.com/janl/mustache.js)
- [Cryptico](http://github.com/wwwtyro/cryptico)
- [W3.CSS](http://w3schools.com/w3css)
